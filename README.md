# 📋 CLI To-Do List App

A terminal-based task manager that lets you add, view, and delete tasks. All tasks are stored in a local file for persistence.

## 🚀 Features
- View all tasks
- Add new tasks
- Delete tasks by number
- Data saved in `tasks.txt`

## 🛠️ Tech Stack
- Python 3

## 🔧 How to Run

```bash
python todo.py
